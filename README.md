# Microservice for sending emails

### Configuração do projeto

- Spring Boot
- Maven
- Java 11

### Dependências

- Spring Web
- Spring Data JPA
- PostgreSQL Driver
- Validation
- Lombok
- Java Email Sender
- RabbitMQ


## Projeto

- Implementação de um microserviço para envio de emails.
- Sistema de comunicação assíncrona com RabbitMQ
